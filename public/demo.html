Демонстрация виджета обратной связи

Как использовать:

Виджет появится в правом нижнем углу. Нажмите на него, чтобы открыть форму обратной связи.

Параметры:

  • data-project-id - ID проекта (обязательно)
  • data-position - Позиция (bottom-right, bottom-left, top-right, top-left)
  • data-theme - Тема (light, dark)
  • data-color - Основной цвет (hex)
  • data-api-url - URL API для отправки данных
